Components

Arduino

We researched the Arduino and the programming it executes. We examined different sensors that measure what's in the and what kind of coding we'd need to build the ground station, as well as the python code needed for the radio receiver we would install inside the can.

Parachute

We investigated different parachute form factors then decided on the classic parachute. The parameters of the parachute were decided using the formulae in the CanSat guidelines. For the materials we found a durable ripstop nylon fabric. Then used kevlar thread to stitch it together

BME280

We decided on using the BME280 because of the accurate measurements in comparison with the basic kit sensors Also because it simplify our construction as it measures barometric pressure, temperature, altitude, and humidity.
