Research

  1. We looked into the Arduino micro-controller and the programming language in which it executes, as well as the many systems that are compatible with the device.
  2. We investigated all the different types of the sensors that measure different metrics and what kind of coding we'd need to implement them into one system.
  3. We then inspected the software we would need to develop the ground station, as well as the C++ code for the radio transmitter we'd install inside the can.
  4. We examined several power sources, such as batteries, and ultimately opted on the lithium ion battery as it is the most efficient and lightest.
  5. We then analysed the aerial that transmits the radio frequencies necessary for sending data back to our workstation. After a bit of deliberation we settled on a wire aerial. Then we found out the frequency in Athlone which is 444 Mhz.
  6. We looked into the numerous types of data that BME280 can provide, such as barometric pressure, temperature, altitude, humidity and also we learned how the gyrcoscope could be used in Arduino.
